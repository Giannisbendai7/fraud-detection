# fraud-detection
This project focuses on building a machine learning pipeline for detecting credit card fraud using real-world transactional data.

The goal is to identify fraudulent transactions by handling highly imbalanced datasets and applying classification algorithms with proper evaluation metrics.

The project includes data preprocessing, feature engineering, model training, and performance evaluation focused on detecting fraud cases accurately.

Tech stack: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

Key steps:

Data cleaning and preprocessing
Handling imbalanced dataset techniques (e.g. class weighting / resampling)
Feature engineering
Model training (classification algorithms)
Evaluation using precision, recall, F1-score, ROC-AUC
